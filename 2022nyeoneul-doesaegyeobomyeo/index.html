2022년을 되새겨보며

Home Network 구성 [2022. 01. 18.]

Router: Ubiquiti UniFi Security Gateway
Switch: Ubiquiti UniFi Switch 8 PoE (60W)
AP: Ubiquiti UniFi U6 Lite

사실상 홈랩의 시발점. UniFi 제품을 우연히 알게 되었는데 외부 디자인과 WebUI가 이뻐서 빠져들게 되었다. 컨트롤러는 DS620slim에 컨테이너로 올려서 굴렸다.

10" 9U Rack 구매 [2022. 01. 24.]

독일에서 직구한 랙. 모든 것을 랙에 채워넣고 깔끔하게 끝내려고 했는데 선반 자체의 크기를 고려하지 않아 DS215j는 들어가지 않고... 공간은 좁아서 정리도 힘들고...

구매한지 일주일도 지나지 않아 19" 랙으로 다시 세팅하자고 마음을 먹었다.

Galaxy Book S 구매 [2022. 02. 11.]

CPU: Intel Core i3-L13G4
RAM: LPDDR4X 4267MHz 8GB
SSD: UFS 3.0 256GB
PSU: 42Wh (25W Charge)
Display: 13.3" 1920x1080 60Hz IPS

통학을 하며 태블릿만 가지고 다녔는데 불편함이 꽤 있어서 가볍기만 한 노트북을 구매했다. 성능을 생각하면 싸진 않지만 가볍다는 것 하나만으로 굉장히 만족스럽다.

성능이 필요한 것들은 원격으로 하면 되기도 하고 말이다.

램은 좀 아쉽긴하지만... 이런 저런 이유보다는 Lakefield를 한번 써보고 싶기도 했다.

NUC 11 Pro 구매 [2022. 02. 12.]

CPU: Intel Core i3-1115G4
RAM: ESSENCORE KLEVV DDR4 3200MHz 16GB SODIMM *2
SSD_1: Samsung PM981 256GB
SSD_2: Transcend 430S 256GB

ESXi를 올려서 사용할 목적으로 구매했다. 부팅 드라이브로 사용하던 USB가 인식이 자주 떨어져나가 고생했던 기억이 있다.

1300 1TB 구매 [2022. 02. 22.]

DS620slim에 넣을 SSD로 구매했다. PLP라는 말에 낚여 구매한 것 같긴한데, 클라이언트 타겟 제품이라 크게 기대하진 않는다.

Elements 16TB 구매 [2022. 03. 19.]

백업을 위해서 구매한 드라이브. 무슨 일이 있더라도 이것만 가지고 도망치자! 라는 생각으로 구매했다. 모으는 데이터의 용량을 제한할 것이라 결심했다는 뜻이기도 하다.

900P 280GB *2 구매 [2022. 03. 31.]

추후 ZFS로 넘어갈 때 SLOG로 사용하려고 구매했다. 

P4800X 375GB 구매 [2022. 08. 16.]

바로 위에서 구매했던 900P 두 개를 추가금을 받고 P4800X 하나와 교환했다. LP브라켓이 없어서 뭔가 못생긴 기분이다.

7300 MAX 3.2TB *2 구매 [2022. 08. 27.]

이것도 위의 900P와 마찬가지로 ZFS에서 Mirror로 쓰려고 구매했다. 

2022년은 전반기에만 달리고 후반기엔 얌전히 돈을 저축하며 지냈다.

서버를 업그레이드 하기 위해서 그랬는데, 아직도 완성하지 못했다.
